# agentic-principles

Always-on principles and user rules for coding agents, copied from
[agentic-cfg](https://github.com/Tsurai7/agentic-cfg)
(`core/principles/` and `core/instructions/user-rules/`).

## Contents

- [principles/honesty.md](principles/honesty.md) — honesty and transparency: back claims with evidence, separate fact from inference, report outcomes faithfully, no hidden actions.
- [principles/karpathy-guidelines.md](principles/karpathy-guidelines.md) — think before coding, simplicity first, surgical changes, goal-driven execution.
- [principles/README.md](principles/README.md) — principle vs skill: where new guidance goes.
- [user-rules/coding-principles.md](user-rules/coding-principles.md) — minimal scope, no over-engineering, existing conventions, comments, useful tests only.
- [user-rules/communication.md](user-rules/communication.md) — how to write responses, code citations, commit and PR descriptions.
